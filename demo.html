Scrollculator

Keep on scrolling :)

Scrollculator is a JavaScript plugin that counts the distance you scrolled on a website.

The plugin has no library dependency, and you can add it to any website.

Each time the page is scrolled the plugin counts the amount of pixels and updates the total distance.

It uses local storage to save the total distance for each user that browses your website.

The distance can be displayed in 3 types of metrics: pixels, metric, inches. This can be changed by the user in the options popup

The user can also close the notification and prevent it from appearing on the next time he browses this website. He can then return it by clearing the browser data, but his total distance will be cleared as well.
